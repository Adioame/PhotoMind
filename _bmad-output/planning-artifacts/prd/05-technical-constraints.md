# 5. 技术约束

## 5.1 技术栈约束

| 类别 | 技术选择 | 理由 |
|------|----------|------|
| 桌面框架 | Electron 35 | 跨平台支持、成熟稳定 |
| 前端框架 | Vue 3 + Composition API | 类型安全，开发效率 |
| UI 组件库 | Naive UI | Vue 3 原生支持 |
| 数据库 | sql.js | 嵌入式、零配置 |
| AI | OpenAI/DeepSeek API | 自然语言处理 |
| 向量模型 | @xenova/transformers (CLIP) | 本地向量计算 |

## 5.2 架构约束

- 数据存储在用户本地，不上传到云端
- iCloud 照片仅在 macOS 上可用
- 向量嵌入在应用层计算，不依赖外部向量数据库
- 采用简单直接的单流程搜索架构（LLM 解析 → 混合搜索 → 结果返回）

## 5.3 资源约束

- 开发资源：单人开发
- 开发时间：MVP 约 1 个月
- 预算：无外部服务费用（LLM API 费用除外）
