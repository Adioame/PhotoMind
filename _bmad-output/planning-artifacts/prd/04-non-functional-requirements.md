# 4. 非功能需求

## 4.1 性能需求

| 编号 | 需求描述 | 目标值 |
|------|----------|--------|
| NFR-01 | 搜索响应时间 | < 3 秒 |
| NFR-02 | 照片加载时间 | < 1 秒（100 张） |
| NFR-03 | 内存占用 | < 500MB（10000 张照片） |
| NFR-04 | 冷启动时间 | < 10 秒 |

## 4.2 安全需求

| 编号 | 需求描述 |
|------|----------|
| NFR-05 | 本地数据加密存储 |
| NFR-06 | LLM API Key 安全存储 |
| NFR-07 | iCloud 访问令牌安全管理 |
| NFR-08 | 用户隐私数据本地处理 |
